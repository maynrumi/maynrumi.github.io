COMPARING MACHINE LEARNING AND TECHNICAL ANALYSIS FOR PREDICTING SHORT-TERM STOCK PRICES

Dissertation Project – Dublin - May 2025

Developed and evaluated machine learning models (SGDRegressor, SVR, RandomForestRegressor, XGBoost) and technical analysis indicators (moving averages, RSI, MACD, Bollinger Bands) to predict short-term stock prices.

Data Processing: Preprocessed 11 years of historical stock data (2014–2025) from FactSet for NASDAQ stocks (AAPL, MSFT, NVDA, AMZN, GOOGL), Singapore stocks (DBS, OCBC, UOB, Singtel, Jardine), and indices (DJIA, FTSE 100), using pandas for cleaning and MinMaxScaler, StandardScaler for normalization.

Feature Engineering: Calculated technical indicators and lagged prices as predictive features, splitting data into 10-year training (2014–2024) and 1-year test (2024–2025) sets with TimeSeriesSplit to prevent look-ahead bias.

Model Optimization: Tuned hyperparameters using GridSearchCV, achieving superior performance with SGDRegressor (R² > 0.99, RMSE as low as 0.77 for NVDA) compared to technical analysis (R² ≤ 0.87).

Deployment: Built and deployed a web application using Streamlit (accessible at maynstockprediction.streamlit.app), enabling real-time stock price predictions for end users.

Key Outcome: Demonstrated machine learning’s significant edge over traditional methods, providing actionable insights for traders to optimize short-term strategies.
